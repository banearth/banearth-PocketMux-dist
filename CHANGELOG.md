# 更新日志

## 0.1.28
- **🎮 本地应用一键打开**：不再手填端口——打开即自动扫描本机正在运行的网页应用（带页面标题），点一下直接玩；⟳ 重扫、手填端口兜底、记住上次玩过的
- **本地应用不再白屏**：代理改为流式转发（首字节立即到手机），大体积游戏页边传边渲染；加载中有可见提示，不再无声空白
- **终端界面彩色显示**：完整还原 TUI 的真实颜色（代码高亮、diff 绿色、粗体等，支持 16/256/真彩色），和电脑上的 wt 一致
- **聊天恢复正常滚动**：撤掉之前的限流窗口/裁剪逻辑，整段对话完整渲染、自然上下滚；顶部保留「查看更早」拉取剩余历史
- **桥自动拉活**：桥进程意外退出（崩溃/被杀）时，控制台看门狗 3 秒内自动重启，手机端不再莫名失联
- **电脑网页端 Ctrl+V 贴图**：截图/复制的图片直接粘贴进输入框，走与 📎 相同的上传管线；纯文字粘贴不受影响
- **连接地址去噪**：设置里的电脑 IP 列表不再显示 WSL/Docker/虚拟机等手机够不到的虚拟网卡地址

## 0.1.27
- **修复终端界面空白**：0.1.26 的后台预取会用大响应堵住桥的消息循环，导致切到终端时订阅消息读不到、终端一片空白。撤掉了会阻塞的预取（本地持久缓存保留，看过的 agent 重开仍然秒开）

## 0.1.26
- **说完了·未读 绿点**：舰队列表里，agent 说完话（变空闲）且你还没点进去看，会显示绿色光晕点 +「✅ 有新回复」，一眼看出该点哪个；点进去自动变已读（灰）。运行中=蓝，等待批准=琥珀
- **后台预取**：其它 agent 有新消息时后台自动下载缓存，点进去秒开，不用再等
- **最后活跃时间修正**：显示最后一条真实消息的时间，不再被后台活动带偏（以前没对话也显示“刚刚”）
- **通知合并**：只在 agent 整个回合说完时发一条通知，不再每条消息都推、太碎（需 APK 0.19）
- **长按消息 →「播放」**：朗读该条消息；去掉了顶部的朗读按钮（需 APK 0.19 的原生 TTS）
- **「我的发言」按 agent 分组 + 倒序**：只显示你发给当前 agent 的历史，最新在底部好点，点一条即可再次发送
- **复制智能化**：有选中就复制选中的，没选中就复制整条；去掉「选取文字」（长按本就能选）
- **「↑ 查看更早的消息」按钮** 回归：主聊天顶部，点开分页历史
